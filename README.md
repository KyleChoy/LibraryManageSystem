# LibraryManageSystem
一个图书馆模拟系统，使用 **Python3** 和 **SQLite** 实现。

根据管理员和读者 ID 的不同提供不同的选单和数据库操作。

# Packages
DBInit：数据库初始化文件；

[PrettyTable](https://github.com/jazzband/prettytable)：打印漂亮的表格。
